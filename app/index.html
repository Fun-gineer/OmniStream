    Keyboard shortcuts

    a - toggle streams list
    d - toggle chat
    1 .. 4 - select stream
    5 .. 8 - show chat corresponding to stream 1..4
    z,x,c - set the number of displayed streams to 1,2,4
    x - toggle between horizontal and vertical sptrt in 2-stream mode
    t,u - mute selected stream, unmute selected stream
    l,m,h,s - change selected stream quality -> low, medium, high, source
